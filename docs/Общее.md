# Общее

Линейка FlashForge Adventurer: **5M** (база), **5M Pro** (5M + зашитый корпус, камера, подсветка, фильтр) и **5X** (IFS, новое железо — см. отдельный раздел [AD5X](AD5X-Старт-и-различия.md)). Все на Klipper, CoreXY, поле 220×220×220.

## Объективные недостатки AD5M и как их обходят

| Недостаток | Решение |
|---|---|
| **Картриджный хотенд** — сопло меняется только в сборе (от ~2 т.р. на распродаже, ~4 т.р. без). | Переход на **Bambu-mod** / K1-мод (сменные дешёвые сопла) — см. [Модификации](Модификации.md). |
| **Нет калёного сопла 0.4** — оригинал из обычной стали. | Замена хотенда на вариант со сменными соплами: **Bambu-mod** (под хотенд Bambu Lab), **Соплямод** или **Conch** (отдельные хотенды) — см. [Модификации → Хотенд](Модификации.md#хотенд). |
| **Хотенд не выше 291 °C.** | Разблокировка MCU головы до 320 °C — см. [Прошивка](Прошивка.md#разблокировка-температуры-хотенда-291--320-c). |
| **Мало ОЗУ (128 МБ)** → ошибка E0017 при множестве мелких перемещений; нельзя ставить тяжёлые плагины (телеграм-бот в KlipperMod). | Частично — [`fix_e0017`](https://github.com/ghzserg/zmod/wiki/Macros#fix_e0017), улучшенное охлаждение ([Ошибки](Исправление-ошибок.md)); радикально — внешний хост. |
| **Нет веб-морды в стоке.** | ZMod / Forge-X / KlipperMod — см. [Прошивка](Прошивка.md). |
| **Штатный обдув средний.** | Кастомные сопла обдува — [Модификации](Модификации.md). |
| **Иногда «битые» файлы при печати по Wi-Fi.** | Проверка MD5 ([Ошибки](Исправление-ошибок.md)), печать с флешки. |

## Отличия 5M Pro от 5M

5M Pro — это тот же 5M, но из коробки укомплектован:
- пассивная **термокамера** (зашитый корпус с дверцей);
- **подсветка** и **веб-камера**;
- доп. сопло **0.6** и более богатый комплект аксессуаров;
- **боковой обдув** модели и выбор режима циркуляции воздуха;
- **фильтры**, датчик TVOC;
- функция **автоотключения**;
- кнопка вкл/выкл прямо с дисплея.

> **Примечание:**
> Базовое железо/электроника у 5M и 5M Pro близки, поэтому часть функций Pro добавляется на 5M программно (перепрошивка «5M → Pro» включает, например, кнопку управления подсветкой на экране). Полноценную термокамеру/фильтр всё равно нужно доделывать физически — см. [Модификации](Модификации.md).

## С чего начать новичку

Краткий порядок после покупки (подробно — в [Калибровке](Калибровка.md)): снять транспортировочные винты → смазать направляющие и выровнять натяжение ремней → автокалибровка → калибровка `rotation_distance` экструдера → выбрать [слайсер](Слайсер-и-профили.md) и откалибровать профиль пластика (температура, поток, ретракт, PA).
