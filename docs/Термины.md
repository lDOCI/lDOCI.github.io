# Термины и сокращения

Если встретили непонятное слово в статьях — оно объяснено здесь.

## Прошивка и электроника
- **Klipper** — система управления печатью («мозги»), на которой работает принтер из коробки.
- **Сток / стоковая прошивка** — заводская прошивка без изменений.
- **Мод** — модификация прошивки (ZMod, Forge-X, KlipperMod), добавляющая функции.
- **Веб-морда (Fluidd / Mainsail)** — управление принтером из браузера с компьютера/телефона. В стоке её нет, появляется с модом.
- **root** — права «суперпользователя» в Linux принтера; нужны, чтобы менять системные файлы.
- **SSH** — удалённый доступ к «внутренностям» принтера (его Linux) с компьютера (например, программой WinSCP).
- **MCU** — микроконтроллер (отдельный «чип-исполнитель»); в голове свой MCU, на плате свой.
- **eMMC** — встроенная память принтера (как SSD), на ней живёт прошивка.
- **U-Boot / FEL** — режимы загрузчика; используются для восстановления «кирпича» (см. [Электроника](Электроника.md)).
- **dropbear** — лёгкий SSH-сервер (вариант патча для root).
- **IFS** — система автоматической смены филамента у AD5X (не путать с AMS у Bambu).
- **TVOC** — датчик качества воздуха (летучих веществ) в 5M Pro.
- **прошить / перепрошить** — записать прошивку (обычно с флешки FAT32).

## Калибровка и механика
- **Карта стола (mesh)** — таблица высот поверхности стола; принтер по ней компенсирует кривизну при печати. Строится автокалибровкой.
- **Автокалибровка / автолевелинг** — автоматическое снятие карты стола и оффсета (запускается с экрана принтера).
- **Z-offset** — точное расстояние сопло–стол на первом слое; настраивается во время печати (см. [Калибровка](Калибровка.md#z-offset)).
- **Тензодатчики** — «весы» под столом; принтер ими щупает стол для калибровки.
- **Input Shaper / шейперы** — компенсация вибраций (резонансов), убирает «эхо»/рябь на стенках. Снимается графиком резонансов (см. [Калибровка](Калибровка.md#input-shaper-компенсация-вибраций)).
- **SCV (square corner velocity)** — скорость прохождения углов; влияет на их чёткость.
- **антивоблинг** — гайки, гасящие влияние кривизны ходовых винтов на ось Z.

## Печать и слайсер
- **Слайсер** — программа, режущая 3D-модель на слои и G-code (OrcaSlicer и др.).
- **G-code** — команды, по которым печатает принтер.
- **PA (Pressure Advance)** — упреждение давления в хотенде; убирает наплывы/недоливы на углах. Калибруется «башней» в OrcaSlicer.
- **Поток (Flow)** — множитель количества выдавливаемого пластика; калибруется под филамент.
- **Ретракт (retraction)** — откат прутка назад при холостых перемещениях, чтобы не было «соплей»/паутины.
- **Z-hop** — приподнимание сопла при перемещениях, чтобы не задевать модель.
- **Cold pull (колдпул)** — чистка сопла: пруток заправляют, прогревают, остужают и выдёргивают с грязью (см. [Печать](Печать.md#засор-сопла-и-cold-pull)).
- **KAMP** — адаптивная карта стола: принтер щупает только зону под моделью, а не весь стол (быстрее).
- **Exclude Objects** — «исключение объектов»: отмена отдельной детали на столе во время печати.
- **Brim (кайма) / Raft (плот)** — дополнительный контур/подложка вокруг/под деталью для лучшего прилипания.
- **Слоновья нога** — расширение нижних слоёв; лечится компенсацией в слайсере (см. [Печать](Печать.md#слоновья-нога-расширенное-основание)).

## Железо хотенда
- **Хотенд** — узел, плавящий пластик (горло + нагреватель + сопло).
- **Сопло-картридж** — у стока сопло меняется только в сборе с хотендом.
- **Bambu-mod / Соплямод / Conch** — разные хотенды на замену картриджному, со сменными дешёвыми соплами. Bambu-mod — под хотенд Bambu Lab; Соплямод и Conch — отдельные хотенды (свои версии на AliExpress). См. [Модификации](Модификации.md#хотенд).
- **PT1000** — термистор (датчик температуры) хотенда, стойкий к высоким температурам.
