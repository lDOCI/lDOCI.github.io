# Сеть и удалёнка

## Подключение к Wi-Fi и Orca

Штатный Wi-Fi работает нормально (на хосте реально ~94 Мбит/с по iperf). Подключение OrcaSlicer к принтеру по сети:
1. Подключите принтер к Wi-Fi.
2. На принтере посмотрите IP: вкладка «Home» → справа снизу значок принтера (иконка «i»).
3. В Orca в профиле принтера откройте значок Wi-Fi.
4. Тип хоста — **flashforge**, в поле IP/URL впишите адрес принтера, нажмите «Тест» → `Connection works correctly`.

> **Примечание:**
> **Статический IP** на принтере часто «не держится» — при включении он всё равно берёт случайный адрес. Надёжнее закрепить IP по MAC на роутере (DHCP-резервация). Для удалённых сервисов нужен реальный IP (`0.0.0.0` не подойдёт) и включённый **LAN mode**.

> **Примечание:**
> На свежих стоковых прошивках Orca-Flashforge / FlashMaker могут показывать принтер **OFFLINE**. Поддержка в этом случае советует ставить новое ПО **FlashStudio** (десктоп) и приложение FlashMaker — с ним принтер виден и по USB, и по сети.

## Проблемы с Wi-Fi

Если связь периодически отваливается:
- Сосед/другая сеть глушит ваш канал — посмотрите Wi-Fi Analyzer и **смените канал** на роутере.
- Слабый Wi-Fi-модуль (чаще на 5M) — подключите по **кабелю Ethernet**, а если тянуть провод неудобно — через **Powerline-адаптеры** (передача сети по электропроводке, комплект от ~1200 ₽).
- Настроить Wi-Fi можно и через **KlipperScreen** (Настройки → Wi-Fi → выбрать сеть → пароль → сохранить).

## Камера по сети

На модах с веб-интерфейсом (ZMod, Forge-X, KlipperMod) поток камеры доступен по `http://IP:8080/?action=stream` и снимок `?action=snapshot`; в Fluidd/Mainsail камера видна сразу. На ZMod порты задаёт макрос `ZSSH_ON` (по умолчанию moonraker `7125`, видео `8080`).

## Удалённый доступ

- **Облако:** Orca-Flashforge + приложение **FlashMaker** ([Orca-FF](https://wiki.flashforge.com/en/Orca-Flashforge-and-Flashmaker/orca-flashforge-quick-start-guide), [FlashMaker](https://wiki.flashforge.com/en/Orca-Flashforge-and-Flashmaker/flashmaker-quick-start-guide)) — привязка по QR-коду.
- **VPN (vless и т.п.):** зайти по SSH на прошитый принтер → установить VPN-клиент → на сервере открыть/маршрутизировать порты Klipper/Moonraker/экрана → с ПК подключиться в ту же VPN-сеть и стучаться в порты.
- **Простой вариант:** перевести принтер на локалку, запретить выход в сеть, а смотреть удалённо через Chrome Remote Desktop с ПК, к которому подключена камера.

> **Совет:**
> Серийник и checkCode принтера лежат в `/opt/config/Adventurer5M.json` (нужны для привязки/восстановления). IP можно получить из `ifconfig`.

## Telegram-бот (moonraker-telegram-bot)

Уведомления о прогрессе/окончании с фото. Ключевое из `telegram.conf`:
```
[bot]
server: 192.168.1.51:7125 # IP принтера : порт moonraker
bot_token: <токен от @BotFather, имя бота обязательно с _bot на конце>
chat_id: <ваш chat_id, бот подскажет при первом сообщении>

[camera]
host: http://192.168.1.51:8080/?action=stream
host_snapshot: http://192.168.1.51:8080/?action=snapshot
type: mjpeg
fps: 15

[telegram_ui]
buttons: [status,pause,cancel,resume],[files,emergency,macros,bot_restart,reboot]
```

## Home Assistant

Принтер интегрируется в HA (есть [черновик-инструкция по установке HAOS](https://gowe.yonote.ru/share/haos-0)). Пример автоматизации «печать завершена + фото» в `automations.yaml`:
```yaml
- alias: Телега печать закончена
 trigger:
 - platform: state
 entity_id: sensor.adventurer_5m_status
 from: BUILDING_FROM_SD
 to: BUILDING_COMPLETED
 action:
 - action: camera.snapshot
 target: { entity_id: camera.adventurer_5m_camera }
 data: { filename: /config/www/cam_captures/done.jpg }
 - delay: { seconds: 3 }
 - action: telegram_bot.send_photo
 data: { file: /config/www/cam_captures/done.jpg, caption: "Печать завершена" }
```
Предварительно настройте telegram_bot в HA. Аналогично делается уведомление на 50% (триггер по сенсору прогресса `above: 50`).

## ZMod: автозакрытие диалогов экрана

`FAST_CLOSE_DIALOGS` закрывает окно на родном экране по окончании/отмене печати (экран не зависает). Параметр `CLOSE_DIALOGS` в `SAVE_ZMOD_DATA` (0 — нет, 1 — медленно, 2 — быстро). Для быстрого закрытия включите на экране: Настройки → значок Wi-Fi → Сетевой режим → «Только локальные сети».
